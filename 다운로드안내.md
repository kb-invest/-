## 🚀 설치 방법 요약

### 1단계: 필수 프로그램 설치
- Node.js: https://nodejs.org (LTS 버전)
- Python: https://www.python.org (Add to PATH 체크!)

### 2단계: 파일 다운로드 및 압축 해제
- `realestate-proposal-source.tar.gz` 다운로드
- 원하는 폴더에 압축 해제

### 3단계: 의존성 설치
```bash
cd [압축해제한폴더]
npm install
cd backend
pip install -r requirements.txt
cd ..
```

### 4단계: 템플릿 파일 추가 (선택)
- `templates/` 폴더에 PPTX 템플릿 복사
  - `template_briefing.pptx` (비크님이 주신 브리핑용)
  - `template_print.pptx` (비크님이 주신 인쇄용)

### 5단계: 실행!
- **간단**: `start.bat` 더블클릭
- 또는 수동으로 백엔드 + 프론트엔드 실행

### 6단계: 브라우저에서 접속
```
http://localhost:3000
```
