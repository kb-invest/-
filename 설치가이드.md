# 📦 상업용 부동산 제안서 생성 시스템 - 설치 가이드

## 🎯 **완전 자동 설치** (3분 완료!)

---

## ✨ **특징**

- ✅ **원클릭 설치**: INSTALL.bat 한 번 실행!
- ✅ **바탕화면 아이콘 자동 생성**
- ✅ **CMD 창 없이 실행** (깔끔한 UI)
- ✅ **자동 의존성 설치** (Python 패키지)
- ✅ **USB 복사 가능** (휴대용!)

---

## 📋 **시스템 요구사항**

### **필수:**
- ✅ **Windows 10/11**
- ✅ **Python 3.8 이상** 
  - 다운로드: https://python.org/downloads
  - ⚠️ 설치 시 **"Add Python to PATH"** 체크 필수!

### **선택 (개발자용):**
- Node.js (프론트엔드 빌드용)

---

## 🚀 **설치 방법** (초간단!)

### **1단계: ZIP 파일 다운로드**

**링크:** https://github.com/kb-invest/-/archive/refs/heads/main.zip

또는

**브라우저에서:**
1. GitHub 페이지 방문
2. 초록색 "Code" 버튼 클릭
3. "Download ZIP" 클릭

---

### **2단계: 압축 풀기**

1. **-main.zip 파일 찾기** (다운로드 폴더)
2. **우클릭 → 압축 풀기**
3. **원하는 위치에 풀기** (예: `C:\부동산앱\`)

---

### **3단계: 설치 실행**

1. **압축 푼 폴더 열기**
2. **INSTALL.bat 더블클릭** ⭐
3. **3분 기다리기** (자동 설치!)

---

### **4단계: 완료!**

설치가 끝나면:
- ✅ **바탕화면에 아이콘 생성됨!**
- ✅ **"부동산 제안서 생성 시스템" 아이콘**

---

## 🎊 **사용 방법** (초간단!)

### **실행:**
```
바탕화면의 "부동산 제안서 생성 시스템" 아이콘 더블클릭!
→ 5초 후 브라우저 자동 열림
→ 앱 사용!
```

### **종료:**
```
설치 폴더의 "종료.bat" 실행
또는
작업 관리자에서 python.exe 종료
```

---

## 📁 **설치 후 파일 구조**

```
C:\부동산앱\               (예시 경로)
├── INSTALL.bat            ← 설치 프로그램 (한 번만 실행)
├── 부동산앱.vbs           ← 실행 파일 (바탕화면 아이콘이 이것 실행)
├── 종료.bat               ← 종료 프로그램
├── backend/               ← Flask 백엔드 서버
├── dist/ 또는 public/     ← React 프론트엔드 (빌드됨)
├── templates/             ← PPTX 템플릿
└── output/                ← 생성된 제안서 저장 폴더
```

---

## 💡 **자동 설치 프로세스**

```
[1/6] 시스템 요구사항 확인
      - Python 설치 확인
      - Node.js 설치 확인 (선택)

[2/6] Python 패키지 설치
      - Flask, python-pptx, Pillow 등 자동 설치

[3/6] 프론트엔드 빌드
      - npm install (Node.js 있는 경우)
      - npm run build (React 빌드)

[4/6] 바탕화면 아이콘 생성
      - "부동산 제안서 생성 시스템.lnk" 생성
      - 자동으로 바탕화면에 배치

[5/6] 실행 파일 생성
      - 부동산앱.vbs (CMD 창 없이 실행)
      - 종료.bat (서버 종료)

[6/6] 설치 완료!
      - 바로 실행 가능
```

## ⚠️ **문제 해결**

### **Python이 없다고 나와요**

**해결:**
1. https://python.org/downloads 방문
2. "Download Python" 버튼 클릭
3. 설치 파일 실행
4. ⚠️ **"Add Python to PATH" 체크 필수!**
5. 설치 완료 후 **PC 재부팅**
6. INSTALL.bat 다시 실행

---

### **바탕화면 아이콘이 안 생겼어요**

**해결:**
1. 설치 폴더 열기
2. `부동산앱.vbs` 파일 찾기
3. 우클릭 → "바로가기 만들기"
4. 바로가기를 바탕화면으로 드래그

---

### **실행이 안 돼요**

**해결 1: 종료.bat 실행**
```
기존 서버가 실행 중일 수 있음
→ 종료.bat 실행 후 다시 시도
```

**해결 2: 포트 확인**
```cmd
netstat -ano | findstr :3000
netstat -ano | findstr :5000

# 해당 프로세스 종료
taskkill /F /PID [PID번호]
```

---

### **API 키 오류**

**해결:**
1. 공공데이터포털 가입: https://data.go.kr
2. 건축물대장 API 신청
3. 승인 대기 (1-2일)
4. **인코딩된 키** 사용 (디코딩 키 아님!)

---

## 🎯 **USB 휴대용 사용**

### **방법:**
1. 설치 완료된 폴더 전체를 USB에 복사
2. 다른 PC에 붙여넣기
3. 해당 PC에 **Python만 설치**되어 있으면 실행 가능!
4. `부동산앱.vbs` 더블클릭으로 실행!

**장점:**
- ✅ 재설치 필요 없음
- ✅ Python만 있으면 어디서든 실행
- ✅ 모든 설정 보존됨 (API 키 등)

## 💬 **자주 묻는 질문**

### **Q: Node.js가 없어도 되나요?**
A: 네! Python만 있으면 실행됩니다. Node.js는 개발자 모드용입니다.

### **Q: 인터넷 연결이 필요한가요?**
A: 설치 시 Python 패키지 다운로드를 위해 필요합니다. 실행 시에는 API 호출을 위해 필요합니다.

### **Q: 다른 PC로 옮길 수 있나요?**
A: 네! USB에 복사해서 다른 PC에서 사용 가능합니다. Python만 설치되어 있으면 됩니다.

### **Q: 업데이트는 어떻게 하나요?**
A: 최신 ZIP 파일을 다시 다운로드하고 INSTALL.bat을 실행하면 됩니다.
