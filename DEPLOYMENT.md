## 🎯 사용 방법

### 1단계: API 키 준비
공공데이터포털(https://www.data.go.kr/)에서 다음 API 인증키를 발급받으세요:
- 건축물대장 정보 조회 API
- 토지대장 정보 조회 API

### 2단계: 기본 정보 입력
1. 웹사이트 접속
2. API 키 입력
3. 지번 또는 도로명 주소 입력
4. 검색 버튼 클릭 → 자동으로 건축물/토지 정보 가져오기

### 3단계: 프로젝트 정보 작성
1. 프로젝트명, 건물명 입력
2. **GenSpark AI 활용**:
   - "AI 프롬프트 가이드" 탭 클릭
   - 주소, 건물명 입력
   - 원하는 프롬프트 복사
   - GenSpark 채팅에 붙여넣기
   - AI가 생성한 전문적인 문구를 복사
3. 입지 및 특징 설명란에 붙여넣기

### 4단계: 추가 정보 입력
- 매각가, 임대료 입력
- 핵심 특징 3가지 작성
- 메인 사진, 내부 사진, 기타 사진 업로드

### 5단계: 제안서 생성
- **TV 브리핑용 생성** 버튼 클릭 → 회사 TV에서 발표할 때 사용
- **인쇄용 생성** 버튼 클릭 → 고객에게 인쇄하여 전달할 때 사용

PPTX 파일이 자동으로 다운로드됩니다! 📥

## 💡 GenSpark AI 프롬프트 활용 예시

### 입력 프롬프트:
```
서울특별시 강남구 대치동 890-12 (테헤란로 418) 위치한 상업용 오피스 빌딩의 
투자 매력도와 핵심 특징을 전문적이고 고급스럽게 3-5문장으로 작성해줘.
```

### AI 생성 결과:
```
테헤란로 중심에 위치한 프리미엄 오피스 빌딩으로, 선릉역 도보 3분 거리의 
초역세권 입지를 자랑합니다. 강남 핵심 업무지구의 대로변에 면해 있어 
탁월한 접근성과 높은 시인성을 확보하고 있으며, 지속적인 관리로 최상의 
컨디션을 유지하고 있습니다. 주변 글로벌 기업들과 금융기관이 밀집해 있어 
안정적인 임대 수요가 보장되며, 향후 자산 가치 상승이 기대되는 
프라임 투자처입니다.
```

이 문구를 제안서에 바로 사용하세요!
