## 🚀 **사용 방법 (초간단 3단계!)**

### **1단계: ZIP 다운로드** (5초)

**링크 클릭:**
```
https://github.com/kb-invest/-/archive/refs/heads/main.zip
```

→ `-main.zip` 파일 다운로드됨!

---

### **2단계: 압축 풀기** (10초)

1. **다운로드 폴더**에서 `-main.zip` 찾기
2. **우클릭 → 압축 풀기**
3. **원하는 위치에 풀기** (예: `C:\부동산앱\`)

---

### **3단계: 설치 실행** (3분)

1. **압축 푼 폴더 열기**
2. **INSTALL.bat 더블클릭** ⭐⭐⭐
3. **3분 기다리기**

**자동으로 진행됨:**
```
[1/6] 시스템 요구사항 확인 (Python 체크)
[2/6] Python 패키지 설치 (Flask 등)
[3/6] 프론트엔드 빌드
[4/6] 바탕화면 아이콘 생성 ✅
[5/6] 실행 파일 생성 (부동산앱.vbs)
[6/6] 설치 완료!
```

---

## 📋 **완전 자동화된 설치 프로세스**

### **INSTALL.bat이 자동으로 처리:**

#### ✅ **1. Python 확인**
- Python 설치 여부 확인
- 없으면 안내 메시지 + 다운로드 링크

#### ✅ **2. Python 패키지 설치**
```
pip install Flask
pip install python-pptx
pip install Pillow
pip install requests
pip install Flask-CORS
```
→ 자동 설치!

#### ✅ **3. 프론트엔드 빌드**
- Node.js 있으면: npm install + npm run build
- Node.js 없으면: 사전 빌드 사용

#### ✅ **4. 바탕화면 아이콘 생성**
```
바탕화면 경로 자동 감지
→ "부동산 제안서 생성 시스템.lnk" 생성
→ 아이콘 자동 배치!
```

#### ✅ **5. 실행 파일 생성**
```
부동산앱.vbs 생성 (CMD 창 없이 실행)
종료.bat 생성 (서버 종료)
```

#### ✅ **6. 완료 메시지**
```
설치 완료!
지금 바로 실행하시겠습니까? (y/n)
```

## ⚠️ **시스템 요구사항**

### **필수:**
- ✅ **Windows 10/11**
- ✅ **Python 3.8 이상**
  - 다운로드: https://python.org/downloads
  - ⚠️ **"Add Python to PATH" 체크 필수!**

### **선택:**
- Node.js (개발자 모드용, 일반 사용자는 불필요)

---

## 🔧 **문제 해결**

### **Python이 없다고 나와요**

**해결:**
```
1. https://python.org/downloads 방문
2. "Download Python" 버튼 클릭
3. 설치 파일 실행
4. ⚠️ "Add Python to PATH" 체크!
5. 설치 완료 후 PC 재부팅
6. INSTALL.bat 다시 실행
```

---

### **바탕화면 아이콘이 안 생겼어요**

**해결:**
```
1. 설치 폴더 열기
2. "부동산앱.vbs" 파일 찾기
3. 우클릭 → "바로가기 만들기"
4. 바로가기를 바탕화면으로 드래그
```

---

### **실행이 안 돼요**

**해결:**
```
1. "종료.bat" 실행 (기존 서버 종료)
2. 다시 바탕화면 아이콘 클릭
```
