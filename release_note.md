# 🏢 상업용 부동산 매각 제안서 자동 생성 시스템 v1.0.0

## 📦 PC 설치용 배포 패키지

25년차 상업용 부동산 전문 중개사를 위한 프리미엄 제안서 생성 솔루션

## ✨ 주요 기능

- ✅ 공공데이터포털 API 연동 (건축물대장, 토지대장)
- ✅ GenSpark AI 프롬프트 가이드
- ✅ 듀얼 PPTX 템플릿 (TV 브리핑용 / 인쇄용)
- ✅ 이미지 업로드 및 관리
- ✅ 원클릭 자동 생성

## 📥 다운로드

**Source code (zip)** 또는 **Source code (tar.gz)** 다운로드

## 🚀 설치 방법

1. 압축 해제
2. Node.js + Python 설치
3. `npm install` 및 `pip install -r backend/requirements.txt`
4. `start.bat` 실행 (Windows) 또는 `./start.sh` (Mac/Linux)

자세한 내용은 `PC설치가이드.md` 참조
