## 💡 **개발자 모드: EXE 빌드 방법**

### **로컬에서 EXE 빌드:**

```batch
cd /home/user/webapp
build-exe.bat
```

**또는:**

```bash
# PyInstaller 설치
pip install pyinstaller pywin32

# 프론트엔드 빌드
npm run build

# EXE 생성
pyinstaller --name="부동산앱_설치" ^
    --onefile ^
    --windowed ^
    --add-data="backend;backend" ^
    --add-data="templates;templates" ^
    --add-data="dist;dist" ^
    installer.py
```

**결과:** `dist/부동산앱_설치.exe`

---

### **GitHub Actions 자동 빌드:**

1. **태그 생성:**
```bash
git tag v2.0.0
git push origin v2.0.0
```

2. **GitHub Actions 자동 실행**
   - Windows에서 자동 빌드
   - EXE 파일 자동 생성
   - Releases에 자동 업로드

3. **결과:**
   - https://github.com/kb-invest/-/releases/v2.0.0
   - `부동산앱_설치.exe` 다운로드 가능
