## 🚀 5분 안에 제안서 만들기

### 1️⃣ API 키 준비 (최초 1회만)

1. [공공데이터포털](https://www.data.go.kr) 회원가입
2. "건축물대장 정보 조회 API" 신청
3. "토지대장 정보 조회 API" 신청
4. 발급받은 인증키 복사

### 2️⃣ 기본 정보 입력 (30초)

```
✏️ API 키: [발급받은 인증키 붙여넣기]
📍 주소: 서울특별시 강남구 대치동 890-12
🔍 [검색] 버튼 클릭
```

→ 건축물/토지 정보 자동 조회! ✅

### 3️⃣ AI로 멋진 문구 만들기 (2분)

1. 상단 **"AI 프롬프트 가이드"** 탭 클릭
2. 주소와 건물명 입력
3. "부동산 특징 분석" 프롬프트 **[복사]** 버튼 클릭
4. GenSpark 채팅 열기 → 프롬프트 붙여넣기 → 전송
5. AI가 생성한 멋진 문구 복사
6. "제안서 작성" 탭으로 돌아와서 붙여넣기

### 4️⃣ 나머지 정보 입력 (2분)

```
🏢 프로젝트명: 상업용 부동산 투자 기회
🏗️ 건물명: 다봉타워
💰 매각가: 2,190억
📌 핵심 특징:
   - 선릉역 초역세권
   - 테헤란로 대로변
   - 프리미엄 오피스
```

### 5️⃣ 사진 업로드 (1분)

- 📸 메인 사진: 건물 외관
- 🖼️ 내부 사진: 로비, 사무실
- 📷 기타 사진: 주변 환경

### 6️⃣ 제안서 생성! (10초)

**용도 선택:**

🖥️ **TV 브리핑용 생성**
- 회사 TV에서 발표할 때
- 화려하고 임팩트 있는 디자인

📄 **인쇄용 생성**
- 고객에게 인쇄하여 전달할 때
- 간결하고 정보 중심 레이아웃

→ PPTX 파일 자동 다운로드! 🎉

## 💡 프로 팁

### GenSpark AI 활용 꿀팁

**상황별 프롬프트 예시:**

1. **입지 분석이 필요할 때**
```
서울특별시 강남구 대치동 890-12 부동산의 입지적 장점을 
교통(지하철, 버스, 도로), 상권, 개발 호재 측면에서 분석해줘.
```

2. **투자 포인트를 강조하고 싶을 때**
```
다봉타워의 투자 하이라이트를 5가지로 요약해줘.
각 항목은 간결하고 임팩트 있게 작성하고, 
투자자 관점에서 매력적으로 표현해줘.
```

3. **수익성을 설명하고 싶을 때**
```
다봉타워 건물의 임대 수익성과 자산 가치 상승 가능성을 분석해줘.
현재 시장 상황과 향후 전망을 포함해서 투자자를 설득할 수 있게 작성해줘.
```

### 이미지 업로드 팁

- **메인 사진**: 건물 전경, 대로변 뷰
- **내부 사진**: 로비, 엘리베이터 홀, 샘플 오피스
- **기타 사진**: 주변 환경, 지하철역, 주요 랜드마크

고해상도 이미지 (1920x1080 이상) 권장

## ❓ 자주 묻는 질문

**Q: API 키가 없어도 사용할 수 있나요?**
A: 네! API 키 없이도 수동으로 모든 정보를 입력할 수 있습니다.
   단, 공공데이터 자동 조회 기능은 사용할 수 없습니다.

**Q: 생성된 PPTX를 수정할 수 있나요?**
A: 네! 다운로드 후 PowerPoint에서 자유롭게 편집 가능합니다.

**Q: 여러 개의 제안서를 동시에 작업할 수 있나요?**
A: 현재는 한 번에 하나씩 작업하시고, 각각 저장하시면 됩니다.

**Q: 생성 버튼을 눌렀는데 다운로드가 안 돼요.**
A: 브라우저 팝업 차단을 해제해주세요.

**Q: 이미지가 제안서에 안 나와요.**
A: 이미지 업로드 후 "✅ 업로드 완료" 메시지를 확인하세요.
