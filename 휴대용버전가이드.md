# 🎒 휴대용 버전 만들기 가이드

## 📦 휴대용(Portable) 버전이란?

USB에 넣어서 어떤 PC에서도 **설치 없이** 바로 실행할 수 있는 버전입니다!

---

## 🚀 자동 생성 방법 (권장)

### 1단계: 파일 다운로드

GitHub에서 앱을 다운로드하세요:
```
https://github.com/kb-invest/-/archive/refs/heads/main.zip
```

압축을 풀면 다음 파일들이 있습니다:
- `build-portable.bat` ← **이 파일을 실행!**
- `start-portable.bat`
- 기타 앱 파일들...

### 2단계: 자동 생성 스크립트 실행

1. **`build-portable.bat`** 파일을 **더블클릭**

2. 스크립트가 자동으로:
   - ✅ Python Portable 다운로드 (11MB)
   - ✅ Node.js Portable 다운로드 (29MB)
   - ✅ 모든 의존성 자동 설치
   - ✅ USB용 폴더 생성

3. **⚠️ 중요**: 중간에 창이 멈추면
   - 현재 폴더의 모든 앱 파일을
   - `portable-app\app\` 폴더에 복사
   - 다시 아무 키나 누르기

4. 완료! (약 10분 소요)

### 3단계: USB에 복사

생성된 `portable-app` 폴더 전체를 USB에 복사하세요!

---

## 💻 사용 방법

### USB를 다른 PC에 꽂고:

1. `portable-app` 폴더 열기
2. **`start-portable.bat`** 더블클릭
3. 끝! 브라우저 자동 오픈!

### 종료하려면:
- 열린 2개의 검은 창(터미널) 닫기

## 📊 휴대용 버전 상세 정보

### 폴더 구조:
```
portable-app/
├── python/           # Python 3.12 Portable (11MB)
├── node/             # Node.js 20.11 Portable (29MB)
├── app/              # 앱 소스 + 의존성 (1GB+)
└── start-portable.bat  # 실행 파일
```

### 크기:
- **압축 전**: 약 1-1.5GB
- **압축 후**: 약 400-500MB

### 장점:
- ✅ 어떤 PC에서도 실행 가능
- ✅ 설치 필요 없음
- ✅ USB에서 바로 실행
- ✅ 관리자 권한 불필요
- ✅ PC에 흔적 안 남음

### 단점:
- ❌ 파일 크기 큼 (1GB+)
- ❌ 최초 생성 시간 김 (10분)
- ❌ USB 속도에 따라 느릴 수 있음

## 🔧 수동 생성 방법 (고급)

자동 스크립트가 작동 안 하면 수동으로:

### 1. Python Portable 다운로드
```
https://www.python.org/ftp/python/3.12.0/python-3.12.0-embed-amd64.zip
```
→ `python` 폴더에 압축 해제

### 2. Node.js Portable 다운로드
```
https://nodejs.org/dist/v20.11.0/node-v20.11.0-win-x64.zip
```
→ `node` 폴더에 압축 해제

### 3. 앱 파일 복사
앱 파일들을 `app` 폴더에 복사

### 4. Python 의존성 설치
```cmd
cd app\backend
..\..\python\python.exe -m pip install -r requirements.txt
```

### 5. Node.js 의존성 설치
```cmd
cd app
..\node\npm.cmd install
```

### 6. 실행 파일 배치
`start-portable.bat`를 최상위 폴더에 복사

완료!

## 💡 팁

### USB 속도 개선:
- **USB 3.0** 사용 권장
- SSD 타입 USB 사용하면 더 빠름

### 압축해서 공유:
```
7-Zip 사용:
우클릭 → 7-Zip → "압축하기..."
압축 포맷: 7z
압축 레벨: 최대
→ 약 400-500MB로 압축됨
```

### 여러 PC에서 사용:
- USB에서 실행해도 됨
- PC로 복사해서 실행하면 더 빠름

## ⚠️ 주의사항

### 포트 충돌:
다른 프로그램이 3000번 또는 5000번 포트를 사용 중이면 오류 발생
→ 해당 프로그램 종료 후 재실행

### 바이러스 경고:
일부 백신 프로그램이 휴대용 버전을 의심할 수 있음
→ 신뢰할 수 있는 파일이니 허용하세요

### Windows Defender:
처음 실행 시 "알 수 없는 게시자" 경고
→ "추가 정보" → "실행" 클릭

---

## 📞 문제 해결

### "Python을 찾을 수 없습니다"
→ python 폴더에 python.exe가 있는지 확인

### "Node.js를 찾을 수 없습니다"
→ node 폴더에 node.exe가 있는지 확인

### "Module not found" 에러
→ 의존성 재설치:
```cmd
cd app
..\node\npm.cmd install
```

### 브라우저가 안 열림
→ 수동으로 열기: http://localhost:3000
