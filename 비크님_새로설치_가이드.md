## 📥 **1단계: 파일 다운로드 (바탕화면에!)**

1. **이 링크를 Ctrl+클릭** (새 탭에서 열기):
   ```
   https://github.com/kb-invest/-/archive/refs/heads/main.zip
   ```

2. 파일 다운로드 시작 (약 60KB)

3. 다운로드 완료 후:
   - **다운로드 폴더** 열기
   - **-main.zip** 파일을 찾기
   - **파일을 바탕화면으로 드래그** (복사)

---

## 📂 **2단계: 압축 해제 (바탕화면에!)**

1. **바탕화면**의 **-main.zip** 파일 **우클릭**

2. **"압축 풀기"** 또는 **"Extract here"** 클릭

3. 압축이 풀리면 폴더 이름 확인:
   ```
   -main
   ```

4. 이 폴더를 **우클릭** → **"이름 바꾸기"**

5. 새 이름 입력:
   ```
   realestate
   ```
   (짧고 간단하게!)

6. **Enter** 누르기

---

## ⚙️ **3단계: 설치 준비**

### **Node.js 설치 확인:**

명령 프롬프트에서:
```bash
node --version
```

버전이 안 나오면:
- https://nodejs.org 접속
- **LTS** 버튼 클릭 다운로드
- 설치 후 **PC 재부팅**

### **Python 설치 확인:**

명령 프롬프트에서:
```bash
python --version
```

버전이 안 나오면:
- https://python.org/downloads 접속
- **Download Python** 클릭
- 설치 시 **"Add Python to PATH" 체크 필수!**
- 설치 후 **PC 재부팅**

---

## 🚀 **4단계: 앱 설치 및 실행**

### **명령 프롬프트 열기:**

1. **Windows 키 + R**
2. **`cmd`** 입력
3. **Enter**

### **설치 명령어 (하나씩 입력!):**

```bash
cd C:\Users\Mkhansteelec\Desktop\realestate
```
**Enter** 누르기

```bash
npm install
```
**Enter** 누르기 → ⏱️ 2-3분 기다리기

```bash
cd backend
```
**Enter** 누르기

```bash
pip install -r requirements.txt
```
**Enter** 누르기 → ⏱️ 1-2분 기다리기

```bash
cd ..
```
**Enter** 누르기
