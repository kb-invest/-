## 📥 1단계: Node.js 설치 (5분)

### 1) 다운로드
1. 이 링크를 **Ctrl+클릭**하여 새 탭에서 열기:  
   **https://nodejs.org**

2. 화면에 큰 버튼 2개가 보입니다:
   ```
   [      LTS      ]  ← 이거 클릭!
   [ Current 버전  ]
   ```

3. **왼쪽 "LTS" 버튼** 클릭!
   - 파일명: `node-v20.11.0-x64.msi` (또는 비슷한 이름)
   - 크기: 약 30MB

### 2) 설치
1. 다운로드한 **`.msi`** 파일 더블클릭
2. **Next** 버튼 계속 클릭 (5번)
3. **Install** 클릭
4. 설치 완료 화면에서 **Finish** 클릭

✅ **완료!** (5분 소요)

## 🐍 2단계: Python 설치 (5분)

### 1) 다운로드
1. 이 링크를 **Ctrl+클릭**하여 새 탭에서 열기:  
   **https://www.python.org/downloads/**

2. 노란색 큰 버튼 클릭:
   ```
   [ Download Python 3.12.x ]  ← 이거 클릭!
   ```

3. 파일 다운로드:
   - 파일명: `python-3.12.x-amd64.exe`
   - 크기: 약 25MB

### 2) 설치 (⚠️ 중요!)
1. 다운로드한 **`.exe`** 파일 더블클릭
2. **첫 화면에서 반드시 체크!!** ⚠️⚠️⚠️
   ```
   ☑ Add Python to PATH  ← 이거 꼭 체크!!!
   ```
3. **"Install Now"** 클릭 (가장 위에 있는 큰 버튼)
4. 설치 진행 (1-2분)
5. **"Setup was successful"** 화면에서 **Close** 클릭

✅ **완료!** (5분 소요)

## ⚙️ 3단계: 앱 실행 (3분)

### 1) 명령 프롬프트 열기
1. **키보드에서**:
   ```

   ```
2. 나타나는 작은 창에 입력:
   ```

   ```
3. **Enter** 키 누르기
4. **검은색 창**이 열림 (명령 프롬프트)

### 2) 폴더로 이동
검은색 창에 이 명령어를 **정확히** 입력:


**Enter** 키 누르기

### 3) Node.js 설치 확인 및 의존성 설치
다음 명령어 입력:

```bash
npm install
```

**Enter** 키 누르기

⏱️ **2-3분 기다리기** (자동으로 설치 진행)

화면에 뭔가 많이 나오면서 설치됩니다.

### 4) Python 의존성 설치

**Enter** 키 누르기

### 5) 앱 실행! 🎉

## ❓ 문제 발생 시

### "npm을 찾을 수 없습니다" 에러
**해결**:
1. Node.js 설치를 다시 확인
2. **PC를 재부팅**
3. 명령 프롬프트를 **새로 열기**
4. 다시 시도

### "python을 찾을 수 없습니다" 에러
**해결**:
1. Python 설치 시 **"Add Python to PATH"를 체크했는지** 확인
2. 체크 안 했다면 Python을 **재설치** (체크 잊지 말기!)
3. **PC를 재부팅**
4. 명령 프롬프트를 **새로 열기**
5. 다시 시도

### 브라우저가 자동으로 안 열릴 때
**해결**:
- 브라우저를 수동으로 열고 주소창에 입력:
  ```
  http://localhost:3000
  ```
