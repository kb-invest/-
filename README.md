# 상업용 부동산 매각 제안서 자동 생성 시스템

> 25년차 상업용 부동산 전문 중개사를 위한 프리미엄 제안서 생성 솔루션

## 🏢 프로젝트 개요

수익형, 사옥용, 투자용 등 상업용 부동산 매각 시 사용할 **전문적이고 고급스러운 제안서**를 자동으로 생성하는 웹 애플리케이션입니다.

### 주요 기능

✨ **공공데이터 API 연동**
- 건축물대장, 토지대장 자동 조회
- 지번/도로명 주소 입력으로 기초 데이터 자동 수집

📄 **듀얼 템플릿 시스템**
- **TV 브리핑용**: 화려한 디자인, 프레젠테이션 최적화
- **인쇄용**: 간결한 표 형식, 출력 최적화

🤖 **GenSpark AI 프롬프트 가이드**
- 부동산 특징/장점을 전문적으로 작성하는 AI 프롬프트 제공
- 맞춤형 프롬프트 생성 및 복사 기능

📸 **이미지 관리**
- 메인 사진, 내부 사진, 기타 사진 업로드
- 제안서에 자동 삽입

💼 **전문가 중심 설계**
- 25년차 중개사의 업무 프로세스 반영
- 고급스럽고 전문적인 디자인

## 🛠️ 기술 스택

### Frontend
- React 18 + TypeScript
- Tailwind CSS (고급 디자인)
- Vite (빌드 도구)
- Axios (HTTP 클라이언트)
- Lucide React (아이콘)

### Backend
- Python 3.x
- Flask (웹 프레임워크)
- python-pptx (PPTX 생성)
- Requests (API 호출)

### External APIs
- 공공데이터포털 건축물대장 API
- 공공데이터포털 토지대장 API

## 📋 사전 준비

### 1. API 인증키 발급

[공공데이터포털](https://www.data.go.kr/)에서 다음 API의 인증키를 발급받으세요:

- **건축물대장 정보 조회 API**
- **토지대장 정보 조회 API**

### 2. 시스템 요구사항

- Node.js 18+ 
- Python 3.8+
- pip (Python 패키지 관리자)

## 🚀 설치 및 실행

### 1. 의존성 설치

```bash
# Frontend 의존성 설치
npm install

# Backend 의존성 설치
cd backend
pip install -r requirements.txt
cd ..
```

### 2. 백엔드 서버 실행

```bash
cd backend
python app.py
```

백엔드 서버가 `http://localhost:5000`에서 실행됩니다.

### 3. 프론트엔드 개발 서버 실행

새 터미널에서:

```bash
npm run dev
```

프론트엔드가 `http://localhost:3000`에서 실행됩니다.

## 📖 사용 방법

### 1. 공공데이터 API 설정

1. 애플리케이션 접속
2. "부동산 기본 정보" 섹션에서 API 키 입력
3. 지번 또는 도로명 주소 입력 후 검색 버튼 클릭

### 2. 제안서 정보 입력

- **프로젝트 정보**: 프로젝트명, 건물명, 입지 설명
- **매각/임대 정보**: 매각가, 임대료, 임대 내역
- **핵심 특징**: 최대 3개의 핵심 특징 입력
- **이미지**: 메인, 내부, 기타 사진 업로드
- **회사 정보**: 중개법인명 및 연락처

### 3. AI 프롬프트 활용

1. "AI 프롬프트 가이드" 탭 클릭
2. 주소, 건물명, 건물 유형 입력
3. 원하는 프롬프트 복사
4. GenSpark 채팅에 붙여넣기
5. AI가 생성한 문구를 제안서에 사용

### 4. 제안서 생성

- **TV 브리핑용 생성**: 회사 TV에서 브리핑할 때 사용
- **인쇄용 생성**: 고객에게 인쇄하여 전달할 때 사용

생성된 PPTX 파일이 자동으로 다운로드됩니다.
